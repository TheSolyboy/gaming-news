Privacy Policy

Last updated: May 21, 2025

1. Overview

This Privacy Policy explains how the Gaming News Bot handles information collected from users on Discord servers where the bot is active.

2. Data Collection

We collect minimal data necessary to provide the bot's core features, which may include:

  • Server and channel IDs (for message delivery)
  • User-submitted content (e.g., via /submit-news)
  • Basic interaction data (e.g., command usage stats)

3. Data Usage

The collected data is used solely to:

  • Deliver gaming news and game deals
  • Moderate submitted content
  • Improve functionality and monitor usage patterns

4. Data Storage

We do not store any personally identifiable information. Data is stored securely and retained only as long as necessary for bot functionality.

5. Third Parties

We do not share data with third parties. Any external links or sources used in news updates are independent of this bot.

6. User Rights

If you wish to have your submitted content removed or have concerns about data handling, please contact the bot owner via the support server.

7. Changes

This policy may be updated from time to time. Continued use of the bot indicates acceptance of the current policy.

8. Contact

For questions or concerns, reach out through the official support server.
